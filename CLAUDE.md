# Spring 학습 프로젝트 — 게시판 → 대규모 서비스

## 학습자 정보

- 웹 백엔드 개발자 지망, 대규모 처리 시스템에 관심
- 자바 기초 문법을 읽을 수 있는 수준, 작은 콘솔 프로그램 3개 작성 경험
- 웹/프레임워크 지식은 없는 상태에서 시작
- 하루 5시간 이상 학습 가능

## 학습 방식

1. **개념 설명** — 해당 Phase의 핵심 개념을 먼저 설명
2. **함께 코딩** — 클로드 코드가 코드를 작성하고, 각 줄의 의미를 설명
3. **직접 도전** — 유사한 기능을 스스로 구현해보는 과제 제시
4. **코드 리뷰** — 작성한 코드를 클로드 코드가 리뷰
5. **리팩토링** — 더 나은 방법이 있으면 함께 개선
6. **문서화** — Phase 완료 후, 학습자가 직접 `docs/`에 문서 작성 → 클로드 코드가 리뷰 (클로드가 대신 작성하지 않음)

## 프로젝트 구조

- 하나의 게시판 프로젝트를 점진적으로 확장하는 방식
- 최종 결과물은 학습 과정에서 결정 (대량 배치 / 이벤트 처리 / 고트래픽 API 중 선택)
